O que é CSS?????

O CSS é uma linguagem de progrmação focada totalmente em estilizar uma pagina na web, conseguindo alterar cores, fontes, estilos e tammbém pode ser usado para organizar uma página

O CSS foi criado a partir de uma necessidade de diminuir o código para estilizar uma página, já que para fazer iso utilizando o Java, era necessário muito tempo para que uma pagina ficasse pronta.
